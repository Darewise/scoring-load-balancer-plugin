Calculates a score of a node depending on preferences for nodes.

There are two preference configurations:

  • Global Preference Score(Preferences for nodes in system global)
  • Project Preference Score(Preferences for nodes in each projects)

The former ones are defined in node configuration pages in "Manage Nodes". The latter ones are defined in project configuration pages.

Each scores is scaled with "Scale for Node Preferences" and "Scale for Project Preferences".
